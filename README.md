# 职航 AI：A02 参考项目

这是一个为 2026“数字马力杯”A02 题目制作的前端参考 MVP。它演示了“职业测评 → 个性化学习计划 → 职场情境训练 → 反馈”的主流程，适合作为比赛原型、答辩演示和后续正式开发的界面基线。

## 已实现

- 职业匹配度总览、能力雷达和本周学习计划
- 可勾选学习任务，完成进度会即时更新
- 三步职业测评弹窗
- 三个可进入的职场模拟训练场景
- 桌面端和手机端自适应布局
- 仅使用本地模拟数据，不会收集或上传任何信息

## 本地运行

需要先安装 Node.js 20 或更高版本，然后在本目录执行：

```bash
pnpm install
pnpm dev
```

浏览器打开终端提示的地址（默认通常是 `http://localhost:5173`）。

构建可部署的静态文件：

```bash
pnpm build
```

构建结果在 `dist` 目录。

## 接到比赛正式项目时怎么扩展

| 当前页面模块 | Java / MySQL 正式实现 | 百宝箱 / AI 能力 |
| --- | --- | --- |
| 职业测评 | Spring Boot 保存题库、答案和用户画像 | 根据测评结果生成职业方向解释 |
| 学习计划 | `learning_task`、`learning_progress` 表与进度接口 | 按目标岗位自动拆解周计划 |
| 场景训练 | `scenario`、`training_session`、`feedback` 表 | 多轮对话、评分反馈、知识库检索 |
| 职业报告 | 报告版本、导出和删除接口 | 生成可解释的能力差距与 3–5 年路径 |

建议先完成本前端的演示闭环，再用 Spring Boot + MySQL 接入真实数据；Python 可作为后续的简历解析、数据清洗或模型调用脚本。若按 A02 正式参赛，还需要在蚂蚁百宝箱企业版中配置智能体、知识库、评测和 MCP / 工具调用，并保留相应截图作为材料。
